# Calculadora Oferta Dojo (Mejorada)

Versión mejorada de la calculadora comercial Dojo.

**Mejoras sobre la original:**
- URL de PWA dinámica (funciona en cualquier dominio, no solo GitHub Pages de luismiguel1977)
- Alquiler de terminales del proveedor configurable (antes fijo a 5€)
- Campo de recargo 0,05€ con texto aclaratorio (poner 0 si el competidor no lo cobra)

## URL en el móvil

https://luismiguel1977.github.io/calculadora-dojo-mejorada/dojo-v2.1.html

## Uso local

```powershell
.\servir.ps1
```

Luego: http://localhost:8080/dojo-v2.1.html
